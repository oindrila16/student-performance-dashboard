**Project Overview**

This project is an interactive data analytics and machine learning dashboard built using Streamlit. It enables users to explore student performance data, visualize insights, generate reports, and predict student scores using a machine learning model.



The project demonstrates skills in data analysis, visualization, machine learning, and web application development.



**Features**



**Authentication**

* Simple login system using username and password
* Access-controlled dashboard



**Data Visualization**

* KPI metrics including average, maximum, and minimum scores
* Bar charts for student performance comparison
* Pie chart for gender distribution
* Subject-wise performance analysis



**Advanced Analytics**

* Correlation heatmap for numerical variables
* Statistical summary of dataset



**Machine Learning**

* Linear Regression model for score prediction
* Predicts student average score based on subject marks



**Report Export**

* Download dataset as CSV file
* Download analysis report as PDF



**Interactive Dashboard**

* Built with Streamlit
* Filter-based interactive data exploration



**Tech Stack**

* Python
* Streamlit
* Pandas
* NumPy
* Plotly
* Matplotlib
* Seaborn
* Scikit-learn
* FPDF
